# Drag.Value

Allows the dragging of things.

# Example

    <input placeholder="Drag a button to me" />

    <button
        data-control="dragvalue"
        data-text-value="Foo">
        Drop "Foo"
    </button>

    <button
        data-control="dragvalue"
        data-text-value="Bar">
        Drop "Bar"
    </button>
