This is the followup of the famous aeurltool extension. When a XSS vulnerability 
was discovered in February 2012 (see
http://typo3.org/teams/security/security-bulletins/typo3-extensions/typo3-ext-sa-2012-001/ )
the extension was discontinued.

dix_urltool is a substitution for aeurltool
