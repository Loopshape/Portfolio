Available expressions
Expressions that can be used to replace content:

  • {*} symbolise the current content
  • {counter} counter when looping around the records
  • {date} date of day
  • {creation_date} creation date of the record - if available.
  • s/search me/replace me/ expression to find and replace.
Expressions can be combined as example:
{counter} - {*} followed by a sample word
