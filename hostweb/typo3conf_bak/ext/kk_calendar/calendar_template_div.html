EXAMPLE





This template features:
- Subparts: date header and date content
- Markers: ###TYPE###: type / Typ (calendar or ToDo)
###WOTA###: Day of week / Wochentag
###DATE###: Date / Datum
###DATETEXT###
###TIME###: Time / Uhrzeit
###HEADER###: Title / Titel
####CONTENT####: Text starting at the second row beyond the time-row
###CATEGORY###: category / Kategorie ###LINK### ###TEXT_CALWEEK### ###CALWEEK### for ToDos: ###WEEK###: nomen est omen
###PRIORITY### ###WORKGROUP### ###RESPONSIBLE### ###COMPLETE### and related text-markers like ###TEXT_....### for all markers above (See the source code for subparts and markers)
